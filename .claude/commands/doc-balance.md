📝 ドキュメント更新チェック

更新前に以下を確認してください：

□ 追記部分が長すぎないか
  → 周囲のセクションと同じくらいの分量か？
  → 本来の優先順位を崩していないか？

□ 追記部分が詳細すぎないか
  → 周囲と同じ粒度で書かれているか？
  → 本来の優先順位を崩していないか？

□ 追記位置は適切か
  → 重要でない内容を上の方に書いていないか？
  → 本来の優先順位を崩していないか？

□ 既存の構造を壊していないか
  → 元々の文書の流れは保たれているか？

□ 不要なナンバリングをしていないか
  → 順序性がない項目に番号を付けていないか？
  → 並列的な内容はリスト形式（-）にすべきでは？
  （番号は項目追加時の振り直しやコンフリクトの原因）

問題があれば修正してからコミット。
