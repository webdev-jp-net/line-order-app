# プロジェクト概要
リアルロケーションとの組み合わせで謎解きをするスマートフォンアプリケーションのLINEアプリ（LIFFアプリ）プロジェクト。

### 基本コンセプト

謎解きプロジェクトは、リアルロケーションを舞台にした周遊型謎解きイベントをデジタル化したWebアプリケーションです。参加者はLINE公式アカウントを通じて謎解きに挑戦し、実際の観光地や施設を巡りながらゲームを進めていきます。

### 共通特徴

- **リアルロケーション連動**: 実際の場所を訪れることで謎解きが進行する
- **開催期間**: 1か月〜数か月間の長期開催を前提とした設計
- **LINE連携**: LINE公式アカウントを活用した参加体験
- **デジタル×リアルの融合**: QRコードやGPS情報を活用した体験設計

## システム構成（共通）

### 外部サービス連携

- **LINE 公式アカウント**: ユーザー認証・通知・共有機能
- **Google Tag Manager**: イベントトラッキング・分析
- **microCMS**: コンテンツ管理システム

## ゲームコンテンツ仕様

ゲームアプリとして提供するコンテンツ

### スタンプラリー

- **内容**: リアルロケーションへ設置されたQRコードを読み取ることで手がかりを獲得できる。
- **形式**: QRコードリーダー起動 + 判定

### 通常謎

- **内容**: 手がかりをもとに挑戦できる謎解き問題
- **形式**: 選択肢の入力 + 正誤判定

### 大謎

- **内容**: これまでの謎の要素を統合した最終問題
- **形式**: 選択肢の入力 + 正誤判定

### オープニングストーリー

- **内容**: 導入ストーリーと最初のチャレンジへの誘導
- **形式**: ストーリー表示 + 動画再生

### エンディングストーリー

- **内容**: 締めのストーリー
- **形式**: ストーリー表示 + 動画再生

### オンボーディング

- **内容**: 初回参加者向けのチュートリアル（基本操作、謎解きのルール、注意事項）
- **形式**: 画面遷移型の説明ページまたはデモプレイ

### プロフィール登録

- **内容**: ユーザーの基本情報登録（年齢性別などの属性、来訪理由）
- **形式**: 入力フォーム

### プレゼント引き換えQRコードもぎり

- **内容**: クリア画面にユニークQRコードを表示し、現地スタッフが読み取ることで重複利用を防止
- **形式**: QRコード表示

### アンケート連携

- **内容**: イベント満足度や訪問場所の感想等を収集
- **形式**: 外部アンケートサービスへのリンク

## 参加レポート機能

運用要件として提供する機能

### 出力内容

- **内容**: 参加日時、正解率、属性情報等
- **形式**: レポート画面表示またはPDF出力

### 出力頻度

- **内容**: リアルタイム更新、日次集計、週次レポート、最終レポート
- **形式**: 管理画面での表示またはレポートファイル出力
