# LINE謎解きゲーム要件定義書

## 案件概要

### 基本情報

- **プロジェクト名**: `line-quest-game`:LINE謎解きゲーム
- **概要**: 謎解きを楽しむLINEログインを活用したスマートフォンアプリケーション。ユーザーは実際の場所を訪れ、現地の情報をヒントに謎を解きながら謎解きを楽しむことができるゲーム。

## 機能仕様

### 認証・初期化

- **LIFF初期化**: LINE Front-end Frameworkの初期化処理
- **LINEログイン**: LINEアカウントによるユーザー認証

**詳細仕様**: [line_auth.md](../common/line_auth.md)

### チュートリアル

- **オープニングストーリー**: ゲーム開始時の導入ストーリー（オープニングストーリー）
- **プロフィール登録**: ユーザー情報の登録・編集
- **遊び方説明**: ゲームの操作方法とルール説明

**詳細仕様**: [tutorial.md](./contents/tutorial.md)

### ナビゲーション

- **アクセスマップ**: 開催地域の地図

**詳細仕様**: [home.md](./contents/home.md)

### 通常謎

- **出題機能**: 各観光スポットでの謎題出題
- **ヒント機能**: 謎解きのヒント表示
- **テキスト回答**: ユーザーの回答入力・判定
- **アクセス情報**: スポット詳細情報とマップ画像
- **結果表示**: 正解・不正解の判定結果

**詳細仕様**: [mission.md](./contents/mission.md#通常謎)

### 大謎

- **最終問題出題**: 全通常謎クリア後の大謎出題
- **テキスト回答**: 最終回答の入力・判定
- **ヒント機能**: 大謎専用のヒント表示
- **結果表示**: 最終問題の正解・不正解判定

**詳細仕様**: [mission.md](./contents/mission.md#大謎)

### プレゼント

- **クリア賞**: 全問正解時の完走プレゼント
- **プレゼント受取**: プレゼント受取場所の案内ともぎり
- **引き換えマップ**: プレゼント受取場所の案内
- **もぎり機能**: 二次元コード読み取り

**詳細仕様**: [prize.md](./contents/prize.md)

### メニュー

- **遊び方**: ゲームの操作方法とルール説明への遷移
- **オープニングストーリー**: オープニングストーリーの再表示機能
- **お問い合わせ**: 外部お問い合わせフォームへのリンク
- **システム情報**: 利用環境情報の表示

**詳細仕様**: [menu.md](./contents/menu.md)
