# 新幹線謎 仕様書

## 概要

新幹線謎（express-mission）は、謎解きシリーズにおいて、新幹線利用者向けに提供される特別版の謎解きアプリです。

## 基本情報

### 用語定義
- **日本語名**: 新幹線謎
- **英語名（コード）**: express-mission
- **用途**: 新幹線版専用謎関連（ルーティング、UI）

### コンセプト
- 新幹線で謎の開催地域へ移動するプレイヤーへの特別なコンテンツを提供することで利用促進する

## コンテンツ仕様

### 問題構成
- **問題数**: 1問のみ
- **出題形式**: 画像による提示、新幹線速度検知により完全な出題内容をアンロックする
- **回答形式**: テキスト入力
- **クリア報酬**: 正解することで新幹線謎専用ノベルティへの応募権利を獲得

### 実現方式
- 外部サービス「VoiStock」を利用して移動速度を計測
- 新幹線乗車相当の速度になると出題を補完する情報を取得可能になりチャレンジできる

### 動作フロー
1. **速度検知の開始**
   - 新幹線謎のコンテンツ内に「速度検知を行う」ボタンを配置
   - ボタン押下で外部サイト「VoiStock」へ遷移

2. **VoiStock での速度検知**
   - VoiStockにて移動速度の計測を実行
   - 新幹線相当の速度を検知した場合、合格判定

3. **アプリへの復帰**
   - 速度検知に合格すると、VoiStockからLIFFアプリへの戻りリンクが提示される
   - 戻りリンクには速度達成条件を満たしたことを示すパラメーターが付与

4. **出題内容のアンロック**
   - アプリに戻ると新幹線謎の完全版の出題内容がアンロックされる
   - 画面には完全版の出題が表示された状態となる

### 解答システムの仕様
- 完全版の出題をアンロックしていない状態でも、解答テキストの送信と答え合わせは可能
- システム側で解答送信をブロックする制限は設けない
- アンロック状態にかかわらず、謎解きへの挑戦自体は可能
