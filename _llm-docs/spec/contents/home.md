# ホーム画面

## 概要

謎解きのメイン画面として、ユーザーが最初にアクセスする画面。
各謎解きスポットへの導線とゲーム全体の概要を提供する。

## ページ情報

- **ルートパス**: `/home`
- **ページコンポーネント**: `Home`

## コンポーネント構成

```
Home/
├── Home.tsx                  # ページコンポーネント
├── useHome.ts               # カスタムフックス
└── Home.module.scss               # スタイル

```

## 機能要件

### 謎解き進捗管理

- **進捗状態の取得**

  - API通信により各謎の解答状態を取得
  - 通常謎と大謎の解答状態を管理

- **表示制御**
  - 解答済み謎の視覚的な区別（チェックマーク、色変更等）
  - 大謎ボタンの活性/非活性制御
  - 条件メッセージの表示切り替え

### ナビゲーション機能

- **内部遷移**

  - 通常謎への遷移（`/mission/[id]`）
  - 大謎への遷移（`/grand-mission`）※条件付き
  - 遊び方ページへの遷移（`/play`）
  - プレゼントページへの遷移（`/prize`）

- **外部リンク**
  - 公式サイトへのリンク（Redux store管理）

### データ取得

- **初期ロード時**
  - ユーザーの謎解き進捗状態をAPIから取得
  - キャッシュ制御による再取得の最適化

## レイアウト要件

### メインビジュアルエリア

- **構成要素**
  - キービジュアル画像
  - ゲーム概要説明文

### 通常謎ナビゲーションエリア

- **表示内容**
  - スポット名
  - サムネイル画像
- **回答状況**
  - 解答済みの場合「クリア」のラベルを画像上に重ねて表示

### 大謎エリア

- **レイアウト**
  - 中央配置のリンクボタン
- **状態別表示**
  謎の解答進捗状況に応じてボタンの活性制御と、ボタン上部に表示するメッセージが変わる
  - 通常謎未クリア時：
    - ボタン：非活性
    - メッセージ：「4つの謎を解いたら押せるよ」
  - 大謎未クリア時：
    - ボタン：活性
    - メッセージ：「最後の謎に挑戦！」
  - 大謎クリア時：
    - ボタン：活性
    - メッセージ：「クリア！おめでとう」

### リンクエリア

- **レイアウト**
  - 内部リンクボタンの水平配置
    - リンクタイトル「遊び方」
    - リンクタイトル「プレゼント」
  - 公式サイトバナーの配置
    - 内部リンクボタンの下に配置
