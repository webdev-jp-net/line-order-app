# メニュー

## 概要

アプリケーション内のナビゲーションとユーザー向け機能を提供するメニュー機能。

## ページ情報

- **ルートパス**: `/menu`
- **ページコンポーネント**: `Menu`
- **説明**: メニュー画面

## コンポーネント構成

### Menu ページ

```
Menu/
├── Menu.tsx                    # ページコンポーネント
├── Menu.module.scss           # スタイル
└── useMenu.ts                 # カスタムフック
```

### MenuCaution ページ

```
MenuCaution/
├── MenuCaution.tsx            # ページコンポーネント
├── MenuCaution.module.scss    # スタイル
└── useMenuCaution.ts          # カスタムフック
```

```

### MenuSupport ページ（システム情報）

```
MenuSupport/
├── MenuSupport.tsx            # ページコンポーネント
└── MenuSupport.module.scss    # スタイル
```

## 機能要件

### カテゴリ管理

- **「もう一度見る」カテゴリ**
  - 遊び方（PlayGuideモーダルを開く）
  - オープニングページへの遷移
  - エンディングページへの遷移

- **「その他」カテゴリ**
  - お問い合わせフォームへの外部リンク

### 外部連携

- **外部リンク管理**
  - Redux store から外部 URL 取得
  - 新しいタブでの外部サイト開封

- **システム情報**
  - システム情報ページ（`/menu/support`）への遷移
  - サポート案内の表示

## レイアウト要件

### メニュー画面

#### カテゴリ構成

##### 「もう一度見る」カテゴリ

| 項目         | リンク/アクション     | 種別       |
| ------------ | --------------------- | ---------- |
| 遊び方       | PlayGuideモーダル表示 | モーダル   |
| オープニング | /menu/story/opening   | 内部リンク |
| エンディング | /menu/story/ending    | 内部リンク |

##### 「その他」カテゴリ

| 項目         | リンク                | 種別       |
| ------------ | --------------------- | ---------- |
| お問い合わせ | Redux store (contact) | 外部リンク |

#### 追加要素

- **システム情報リンク**: `/menu/support` へのテキストリンク
- **公式サイトバナー**: Redux store (website) への外部リンク

## 関連ページ

### システム情報ページ（`/menu/support`）

#### ページ情報

- **ルートパス**: `/menu/support`
- **ページコンポーネント**: `MenuSupport`
- **説明**: サポート用システム情報表示

#### 機能要件

- **システム情報表示**
  - USER ID（LINE User ID）
  - OS（`liff.getOS()`）
  - LINE バージョン（`liff.getLineVersion()`）
  - User Agent

- **サポート機能**
  - システムサポートへの外部リンクボタン
  - 全ての表示項目をクリップボードに一括コピー
  - メニューページへの戻るボタン

## 外部リンク管理

Redux store で管理される外部 URL（`external.ts`）:

| 項目          | 用途                 |
| ------------- | -------------------- |
| website       | 公式サイト           |
| contact       | お問い合わせフォーム |
| systemSupport | システムサポート     |
