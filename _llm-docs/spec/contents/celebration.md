# イベントクリア

## 概要

大謎クリア後に表示されるイベント完了画面。

## ページ情報

- **ルートパス**: `/celebration`
- **ページコンポーネント**: `Celebration`
- **説明**: イベントクリア画面

## コンポーネント構成

```
Celebration/
├── Celebration.tsx           # ページコンポーネント
├── useCelebration.ts         # カスタムフック
└── images/                   # 画像アセット
```

## 機能要件

- クリアメッセージの表示
- プレゼント画面への導線
- X（SNS）シェア機能


- **プレゼント画面への遷移**
  - 全問クリア賞応募画面（`/prize`）へ遷移

- **SNSシェア**
  - X投稿画面への下書きつき外部リンク
  - 新規タブで開く

### データ取得

- Xシェアリンク: source.jsonから取得

## アクセス制御

### アクセス条件

- **必須条件**（いずれも満たしていること）
  - 大謎（grand-mission）クリア済み
  - オンボーディング完了済み

### ガード処理

- ContinueGuardにてゲームクリア状態を確認
- 未クリア時は`/`へリダイレクト

## データソース

### source.json

```json
{
  "tweetUrl": "https://example.com"
}
```

## セキュリティ要件

### 外部URL連携

- URL検証: https://スキームのみ許可
- XSS対策: window.open時に`noreferrer`を指定
- 入力値検証: source.jsonの値を使用（ユーザー入力なし）
