# プレゼント

## 概要

クリア後にもらえるノベルティグッズの紹介と受取導線のコンテンツ。

## ページ情報

- **ルートパス**: `/prize`
- **ページコンポーネント**: `PrizeIndex`
- **説明**: プレゼント情報一覧画面

## コンポーネント構成

```
PrizeIndex/
├── PrizeIndex.tsx              # ページコンポーネント
├── usePrizeIndex.ts            # カスタムフック
└── components/
    └── PrizeCard/              # プレゼントカード
        └── PrizeCard.tsx
```

## 機能要件

### プレゼント管理機能

- **進捗状態取得**
  - API通信により各プレゼントの達成状態を取得
  - EX賞と全問クリア賞の応募条件チェック

- **状態表示制御**
  - 各プレゼントの応募条件達成状態の表示
  - アンケート回答済みフラグの管理
  - プレゼント引換済みフラグの管理

### アンケート連携機能

- **外部フォーム連携**
  - 外部アンケートサービスとの連携
  - 回答完了後に本アプリへリダイレクト

- **回答完了処理**
  - アンケート回答済みフラグの更新
  - プレゼント応募状態の再取得

### 二次元コード読み取り機能

- **スキャナー起動**
  - LIFFの二次元コードスキャナーの起動
  - 読み取り成功/失敗の処理分岐

- **引換処理**
  - 二次元コード読み取り後のプレゼント引換処理
  - 引換完了後の状態更新

## レイアウト要件

### プレゼント一覧表示


### プレゼントカード要素

- タイトル（プレゼント名）
- 説明文（プレゼントの詳細説明）
- プレゼント画像（イメージ画像）
- 応募方法（応募手順の説明）
- 引き換え場所画像（受取場所の地図や写真）
- アンケート回答フォームへの導線リンク
- 二次元コード読み取りボタン

### 状態表示

- 条件達成：「条件達成」ラベル
- 条件未達成：「条件未達成」ラベル
- 応募済み：「引き換え済み」ラベル

### ボタン活性化制御

#### アンケート回答ボタン
- 活性化：応募条件達成
- 非活性化：応募条件未達成、またはアンケート回答済み

#### 二次元コード読み取りボタン
- 活性化：アンケート回答済み
- 非活性化：アンケート未回答、またはプレゼント引換済み

### クリア賞（complete）

- **受取条件**: 大謎に正解

## 関連ページ

### プレゼント引換コード読み取り結果画面（`/prize/exchange/:id`）

#### ページ情報

- **ルートパス**: `/prize/exchange`
- **ページコンポーネント**: `PrizeExchange`
- **説明**: プレゼント引換コード読み取り結果画面

#### 機能要件

- ページマウント時に二次元コードスキャナーを自動起動
- スキャン結果の照合
- 照合結果に応じて成功/失敗UIを表示

#### 成功時の表示

- スキャン成功メッセージ
- プレゼント引換の案内
- プレゼント引換ボタン
  - クリック時：プレゼント引換済みフラグ更新
  - 処理完了後：プレゼント一覧画面へ遷移
  - 処理中：ローディング表示

#### 失敗時の表示

- スキャン失敗メッセージ
- 再試行の案内
- プレゼント一覧画面へ戻るボタン
