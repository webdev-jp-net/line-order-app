# プロフィール登録フォーム仕様

## 概要

謎解き挑戦前にユーザーの基本情報を収集するフォーム

## 関連ファイル

- **API設計書**: [`/_document/_llm-docs/spec/fuji/api-structure.yaml`](../api-structure.yaml)
- **定数定義**: [`/src/constants/profile.ts`](../../src/constants/profile.ts)
- **実装**: [`/src/pages/_onboarding/Profile/`](../../src/pages/_onboarding/Profile/)

## フォーム項目

### 性別

- **必須**: ✓
- **表示の種類**: 単一選択

| value | label  |
| ----- | ------ |
| 0     | 女性   |
| 1     | 男性   |
| 2     | 無回答 |

### 年代

- **必須**: ✓
- **表示の種類**: プルダウン

| value | label    |
| ----- | -------- |
| 0     | 9歳以下  |
| 1     | 10代     |
| 2     | 20代     |
| 3     | 30代     |
| 4     | 40代     |
| 5     | 50代     |
| 6     | 60代以上 |

### お住まいの地域

- **必須**: ✓
- **表示の種類**: プルダウン
- **選択肢**: JIS X0401の都道府県コード（01~47）

## API仕様

### エンドポイント

```
PUT /api/profile
```

### リクエストボディ

```json
{
  "userProfile": {
    "gender": 0,           // 0 | 1 | 2
    "ageGroup": 0,         // 0 | 1 | 2 | 3 | 4 | 5 | 6
    "residence": "13"     // JIS X0401都道府県コード（文字列）
  }
}
```

## 実装上の注意

- すべての項目が必須
- 未入力の場合は送信ボタンをdisabled状態にする
- 都道府県コードは2桁のゼロパディング文字列（例: "01", "13"）
- フォーム送信後はHOMEページへ遷移
